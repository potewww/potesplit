Amministrazione

Le funzioni qui sotto richiedono il tuo token amministratore (diverso dai token dei singoli eventi): permette di creare nuove repository GitHub e attivare Pages su di esse. Non condividerlo con nessuno.

⚙️ Token amministratore GitHub
Consigliato: un fine-grained token con Resource owner = la tua organization, Repository access = "All repositories", permessi Administration (org) + Contents + Pages in read/write. Così il token non può toccare i tuoi repository personali, solo quelli dell'organization. Vedi il README per i dettagli (e per l'alternativa più semplice ma meno sicura col token classico). Resta salvato solo in questo browser.
⚠️ js/config.js non è ancora stato configurato con owner/repo/org: finché resta così, la creazione di nuovi eventi non funzionerà. Vedi il README.
